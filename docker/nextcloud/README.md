# GeorgeOS Nextcloud

Diese Nextcloud-Instanz ist die zentrale Dateiablage für GeorgeOS.

Zweck:
- zentrale Ablage für Dokumente
- Inbox für neue Dateien
- Grundlage für Geräte-Synchronisation
- spätere Anbindung an GeorgeOS, KI und MCP

Erste Testversion:
- Docker Compose
- Port 8080
- SQLite als Startlösung

Später geplant:
- PostgreSQL oder MariaDB
- HTTPS
- Domain
- Backup
- Benutzer- und Rechtekonzept
